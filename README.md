Responsive Snake Game
====================

A simple Snake game built with HTML, CSS, and JavaScript. The game is responsive and works well on both desktop and mobile devices.

How to Play
-----------
- Use the arrow keys (or swipe on mobile) to control the direction of the snake.
- The objective is to eat the red food that appears on the canvas, which makes the snake grow.
- The game ends if the snake collides with the wall or itself.
- Once the game ends, click the "Restart Game" button to play again.

Game Controls
-------------
- Desktop:
  - Arrow Up → Move the snake up
  - Arrow Down → Move the snake down
  - Arrow Left → Move the snake left
  - Arrow Right → Move the snake right

- **Mobile:**
  - Swipe up, down, left, or right to change the snake's direction.

Technologies Used
-----------------
- HTML5
- CSS3
- JavaScript
